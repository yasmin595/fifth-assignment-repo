Question-1: What are the different ways to select an element in the DOM?

There are the different ways to select an element in the dom .For example getElementById,querySelectorAll, querySelector, getElementsByClassName, getElementsByTagName.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHtml returns all text, including html tags. textContents is all text contained by an element and all its children that are for formatting purposes only. innerText returns all text contained by an element and all its child elements.

Question-3: What is event delegation in the DOM?

Event Delegation is basically a pattern to handle events efficiently. Instead of adding an event listener to each and every similar element, we can add an event listener to a parent element and call an event on a particular target using the .target property of the event object

Question-4: What is event bubbling in the DOM?

It allows parent elements to respond to events triggered by their child elements. it works Execution Order, Listener Behavior,Event Triggering,Event Capturing,Event Bubbling.

Question-5: How do you create, add, and remove elements using JavaScript?

To create an element in JavaScript,I use document.createElement() and set its content using properties like textContent or innerHTML. To add the element to the webpage, use appendChild() to add it at the end of a parent element or insertBefore() to place it before another element. To remove an element, use removeChild() on its parent or call remove() directly on the element.
